Wall Mosaic — Help & User Guide

Wall Mosaic helps you design picture wall layouts using real-world dimensions—so you can plan everything before you start hanging frames.

Overview

Wall Mosaic lets you:

Create a wall from dimensions or a photo
Add framed pictures with real sizes
Arrange and align layouts visually
Group and lock placements
Generate an installation guide for hanging
Getting Started
1. Create a Wall

Click Add Wall and choose one of the following:

Simulated Wall
Enter wall width and height
No photo required
Corners are automatically defined
Photo Wall
Import a photo of your wall
Drag the corner handles to match the wall edges
Enter real-world wall dimensions
2. Add Pictures

Click Add Picture and:

Enter width and height
Set the hanging hook position (distance from top of frame)

Optional:

Import a photo of the picture for calibration
3. Place Pictures on the Wall
Right-click a picture
Select Place on Wall

The picture will appear centered on the wall.

You can:

Drag to reposition
Select to bring to front
Designing Your Layout
Move & Arrange
Drag pictures to position them
Use the grid to guide placement
Alignment Tools

Align selected pictures:

Left / Center / Right
Top / Middle / Bottom
Grouping
Add pictures to a group
Move them together as a unit
Use alignment tools on the group
Locking
Lock a picture to prevent accidental movement
Locked pictures can still be part of a group
Duplicate Walls
Duplicate a wall to try alternate layouts
Option to include or exclude existing picture placements
Grid Control
Drag on the wall to adjust grid offset
Helps align layouts relative to real-world reference points (e.g., ceiling, furniture)
Installation Report

Generate a report to help you hang your layout:

File → Installation Report…

The report includes:

Wall overview with picture positions
Frame dimensions
Hanging hook offsets
Exact placement measurements:
Distance from left edge
Distance from right edge
Height from bottom of wall
Tips for Best Results
Use accurate wall and frame measurements
Calibrate photo walls carefully for better precision
Lock finished placements before adjusting others
Duplicate walls to explore variations without losing your original layout
Requirements
macOS 14 or later
Privacy

Wall Mosaic:

Does not require an account
Does not collect personal data
Stores all project data locally on your device
Only accesses photos when you choose to import images
